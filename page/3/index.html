ACS 客製化降噪耳塞

Author published on 2024-07-20 included in 降噪耳塞

上面這圖不用多說了吧~ 就是 ACS 客製化耳塞拿到的盒子，滿意外沒有很大XD

想說網路上很少人在分享這東東，就來分享一下唄。

那為什麼會想客製化耳塞呢？ 主要是因為朋友會揪一些 hardstyle 的夜店場跟音樂節像是 Defqon.1、Intents、… 之類的，然後耳膜可是非常脆弱的，我還不想要這麼快就失聰QQ

不過還有另一個原因，可能會佔據比較大部分啦，就是我有時候會需要在網路機房內操作設備，有時候不小心就待很久了，然後高頻風扇又超吵… 基於以上這些原因，就想找個保護的方法摟～

其實可以不需要搞到客製化，直接使用現成的產品即可，但想說都要花錢了，就來體驗一下客製化也不錯，那就順便分享一下我怎麼買的吧。

挑選款式

首先找到官網，這邊連結幫你直接導到客製化的 Pro 17，因為我的 base 就是選擇這一款。

我是選擇全罩款，因為看說明是說可以戴頭盔不被影響，也不知道我怎樣才會有這需求就是了XD
這邊推薦手把款，因為穿脫會比較方便，全罩款的穿脫上會有點點不方便。

接下來就選擇台北 ACS 取模後就可以加入購物車了。

另外建議加購頸繩 ，這真的是有夠方便，脫掉後可以直接垂在脖子不用擔心不見或掉了，頸繩還蠻牢固的，不小心扯一下也不太會掉。

剩下的個性化選項像是顏色或是雷雕啥的就看個人想不想加購摟。
訂單會先到台北 ACS ，所以之後需要再加購什麼可以在製作耳模的時候再做額外加購也行。

選擇完就可以在網頁的購物車頁面上結帳了，我是使用 LinePay 結帳的，有電子支付讚讚！

預約耳模製作

下單完後我是去私訊 ACS Facebook 的官方帳號，也可以打電話啦～

整個製作時長順利的話就 30 分鐘以內即可，前面會先跟你說明介紹一下，之後就開始灌耳漿?!

過程中的感覺滿酷的，就冰冰涼涼的，然後幾秒後填充物就開始膨脹起來，塞好塞滿(,,・ω・,,)

大概過 5 分鐘乾掉後就拆下來，大概長成這樣

其實我到這一步結束後有再加購耳塞的顏色XD 本來挑透明，但是店員說久了可能會變黃黃的，畢竟是塑膠 (這是免費換顏色誒XD)
顏色的話現場都有樣品可以看，有半透明、添加亮粉、… 之類的樣式可以參考。

最後跟店員確定整個訂單的細項後就可以送給英國那邊去製作了，然後等待 4 ~ 5 週的時間就可以寄到手中。

Halo

Author published on 2024-05-14

開場要打招呼，就突然想到 sub zero project 有一首歌叫 HALO，蠻喜歡的推薦推薦XD

第一個貼文就送給 hugo 吧！

首先，不得不說偶的運氣真的差的可憐… 看了網路很多文章都寫說 hugo 5分鐘就可以架設好了，結果窩花了一天的時間(20小時以上) 才搞定開始寫這則貼文…

至於原因嘛… 一開始就挑酷酷的主題去使用，結果呢幾乎每個主題都只說到 git submodule 安裝這邊就結束了，剩下的就是請自己想辦法，對於懂的人來說這倒是真的，但是對於一開始的超級小白就不好說了，尤其是不會去看 Document 的人，這邊就是我盒盒盒。

窩遇到的問題其實很簡單，看著錯誤訊息想了老半天才發覺，就是在設定檔內少了或打錯 theme = 'xxx' 這東東= =

把這個設定加上去後的確 5 分鐘可以搞定，但是還不包含整體環境設定的修改，因為主題的作者都叫你去 exampleSite 直接下指令跑起來。 這就是第二個問題，小白不會一開始就知道 config.toml (現在改成 hugo.toml) 是怎麼搞的，要吃哪裡的設定檔啥的…

大致這兩點問題就是不看 Document 的人，照著說明直接坐下去有機會遇到的吧，事後還是要慢慢的去看一下 Document 唄！ 但是窩就是懶，而且窩其實超級不喜歡網頁相關的東西，但是窩很崇拜寫網頁的人，因為網頁相關的工具技術太多且更新太快，窩根本跟不上…

B.T.W. GitHub Page (GH) 也就是 github.io 這個專案中，只能放 public 裡面的東西，因為 GH 會去讀取該專案下的 index.html 檔。 如果是將整個 hugo push 上去的話會跑不動，就算你加了 .nojekyll 讓 deploy 成功也一樣會得到 404 的訊息盒盒盒 ( 窩不會說這是偶遇到過的)
